# C2048

Simple, quit buggy, [c2048](https://github.com/helgefmi/c2048) AI ffi binding for ruby.

## Installation

Add this line to your application's Gemfile:

    gem 'c2048'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install c2048
